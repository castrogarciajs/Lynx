# Tools For python

We use `ruff` tools for python build with rust, super fast...

## Install ruff

You can install ruff local enviroment with

```sh
pip install ruff
```

## Use

Ruff this configuration for use with `pipenv` in the file `ruff.toml` you can see our code nomenclutra

## Linter

#### Run linter

```sh
ruff check .
```

## Formatter

#### Run formatter

```sh
ruff fmt .
```
